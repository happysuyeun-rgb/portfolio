온프레미스 생성형 AI 업무 도우미

GPT기반 인공지능 토탈 솔루션 메스와이저

본 프로젝트에서 역할은 브랜딩 & Product Design 총괄이었습니다.

#UX Strategy #서비스 브랜딩 #Prompt Engineering #Product Design #Design System

Project Overview

  • On‑premise 환경의 생성형 AI를 적용, 내/외부 데이터로 안전하게 답변하는 메신저형 업무 도우미
  • 업무 문서·대화·지식 DB를 연결하는 프롬프트 체계 및 인텐트 라우팅 설계
  • 민감정보 비유출/감사추적을 고려한 UX 가드레일과 피드백 루프 적용
  • 툴: Figma, OpenAPI 스키마, Python 샘플 스크립트

Problem

  • 사내 검색은 파일명 중심이라 ‘무엇을 찾아야 할지’부터 막히는 경우가 잦음
  • 클라우드 접근 제한으로 외부 LLM 서비스 활용이 어려움
  • AI 응답의 신뢰/출처 표기가 부족해 실제 업무 적용이 제한

Outcome

IA / 검색 UX

  • 챗 기반 질의 → 관련 문서 스니펫 → 근거 출처 표기까지 한 화면 제공
  • 업무 도메인별 프롬프트 탬플릿과 금지어/우회어 사전 정의
  • 에스컬레이션(사람에게 묻기)로 막다른 길 최소화

시각체계 / 확장성

  • 결과 카드 컴포넌트 표준화(요약/근거/후속작업 버튼)
  • 지식 그래프 확장 전제의 토큰/레이블 시스템
  • 감사 로그·승인 플로우 등 엔터프라이즈 거버넌스 요소를 UI에 노출

Reflection

  • ‘정답’보다 ‘근거가 보이는 답’에 대한 만족도가 높았음
  • 모델 품질 개선만큼 데이터 품질/권한 모델링이 중요
  • 사내 배포 문턱을 낮추기 위해 설정/가이드/샘플을 UI에 통합

Impact & Value

  • 사내 검색 대비 답변 시간 단축, 중복 문의 감소
  • 보안/거버넌스를 충족하는 GenAI 도입 레퍼런스 제공
  • 도메인 추가 시 재학습 없이 프롬프트 레벨 확장이 가능
