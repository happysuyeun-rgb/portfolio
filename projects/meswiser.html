Project Overview

The Challenge

배경 및 문제점

  • 사내 검색은 파일명 중심이라 '무엇을 찾아야 할지'부터 막히는 경우가 잦음
  • 클라우드 접근 제한으로 외부 LLM 서비스 활용이 어려움
  • AI 응답의 신뢰/출처 표기가 부족해 실제 업무 적용이 제한

The Solution

핵심 해결 방향

  • 챗 기반 질의 → 관련 문서 스니펫 → 근거 출처 표기까지 한 화면 제공
  • 업무 도메인별 프롬프트 탬플릿과 금지어/우회어 사전 정의
  • 민감정보 비유출/감사추적을 고려한 UX 가드레일과 피드백 루프 적용

Step 01

데이터 유출 없는 보안 질문 (On-Premise)

외부 유출 걱정 없이 사내 데이터를 학습한 AI에게 질문하세요. 기업 보안 규정을 준수하는 안전한 환경을 제공합니다. 클라우드 접근 제한으로 외부 LLM을 쓰기 어려운 기업에 최적화했습니다.

Step 02

사실 기반의 추론 (RAG Engine)

AI가 멋대로 지어내지 않습니다(Hallucination Free). 사내 문서와 매뉴얼을 실시간으로 탐색하여 정답의 근거를 찾아냅니다. 챗 기반 질의 → 관련 문서 스니펫 → 근거 출처 표기까지 한 화면에 제공합니다.

Step 03

출처가 명확한 답변 (Smart Citation)

답변과 함께 참조한 문서의 원본 링크와 페이지까지 정확하게 표기하여, 업무에 즉시 활용 가능한 신뢰도를 보장합니다. 결과 카드 컴포넌트 표준화(요약/근거/후속작업 버튼)로 '근거가 보이는 답'에 집중했습니다.

The Impact — 주요 성과

Zero Hallucination

거짓 답변 최소화로 업무 신뢰도 확보. 보안/거버넌스를 충족하는 GenAI 도입 레퍼런스 제공

Work Efficiency

사내 검색 시간 50% 이상 단축. 답변 시간 단축, 중복 문의 감소

Data Governance

데이터 권한 및 감사 로그 완벽 대응. 감사 로그·승인 플로우 등 엔터프라이즈 거버넌스 요소를 UI에 노출
