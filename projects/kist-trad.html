KIST 전통문화혁신 플랫폼

전통문화 분야의 지식·데이터·전문가·수요처를 연결하는 허브 플랫폼을 설계했습니다. R&D 과제 발굴과 연계를 돕는 탐색 경험을 중심으로, 공공 포털의 접근성 기준을 충족하면서도 현대적이고 직관적인 서비스 UI를 구축했습니다.

#PublicPlatform #UX Strategy #System Architecture #Product Design #Design System

Project Overview

  • 역할: Product Planning · System Architecture · Product Design 총괄
  • 핵심기능: 통합 검색/분야 필터, 중점분야 큐레이션, FAQ/절차 안내, 관련사업 연계, 수요 접수
  • Site Concept: “전통의 미장센 × 모던 UI” — 동양화 일러스트와 현대적 컴포넌트의 조화
  • Tools: Figma, FigJam(플로우/IA), 운영 가이드

Problem

  • 기관·학계·현장 정보가 분절되어 전문가/데이터/지원사업을 한 곳에서 찾기 어려움
  • 전통문화 용어 체계가 난해해 비전문가의 진입 장벽이 높음
  • 수요 접수/지원 절차가 복잡하고 핵심 안내 정보가 분산
  • 공공 포털 기준을 지키면서 브랜드 일관성을 유지하기 어려움

Outcome

1. IA/검색 UX 개선
  • 키워드 중심 통합 검색 및 주제/분야/기관 다층 필터 설계
  • ‘중점분야’ 섹션으로 현안 이슈 큐레이션과 결과 요약 카드 제공
  • FAQ/절차 안내를 질문 기반 인터랙션으로 단순화
2. 시각 체계/확장성 향상
  • 전통 일러스트와 모던 UI를 결합한 브랜드 톤앤매너 정립
  • 카드/리스트/배너/폼 등 컴포넌트 시스템 제공
  • 사업/데이터 모듈 추가를 고려한 확장형 레이아웃 구성

Reflection

  • 도메인 언어를 사용자 언어로 번역하는 IA 카피라이팅의 중요성 확인
  • 전통적 미장센과 현대적 UI의 균형으로 브랜드 경험 연속성 확보
  • 운영 요소(상시 접수/질문 응대)를 고려해 운영 친화적 UX로 설계

Impact & Value

  • 분산된 정보를 허브화하여 탐색 비용을 감소, 참여 전환율 향상
  • Plain Language 기반 안내로 비전문가 진입 장벽 낮춤
  • 기관·전문가·수요처 간 연결 성공 사례 창출
  • 공공 플랫폼에서도 브랜딩 일관성과 확장성을 갖춘 UX 레퍼런스 확보
