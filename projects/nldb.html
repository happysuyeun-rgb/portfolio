브랜딩 · 공공데이터 포털

공공 데이터를 개방하고, 지식과 정보의 흐름을 연결하는 플랫폼. 공공기관의 접근성/보안 기준을 충족시키면서도 사용자가 데이터를 '탐색'하는 경험을 중심으로 UI 패턴을 설계했습니다.

Project Overview

The Challenge

배경 및 문제점

  • 방대한 텍스트 중심 화면 → 사용자 탐색 피로도 증가, 탐색 경로가 단절
  • 전문 용어와 구조가 개발자 중심 → 일반 사용자/연구자에게 난해
  • 공공기관 접근성 가이드로 현대적이고 직관적인 UI를 구현하기 어려움

The Solution

핵심 해결 방향

  • Site Concept: 데이터 탐색을 쉽게, 정보 구조를 명확하게 — 민간 서비스 수준의 사용성을 갖춘 공공 데이터 포털
  • 의도 기반 자동완성/추천 검색 UX와 핵심 메타데이터 요약 카드 도입
  • Tools: Figma (디자인 시스템, 컴포넌트 문서화)

Step 01

데이터의 의미론적 연결 (LOD)

단순한 정보 축적을 넘어, 데이터 간의 관계를 정의하고 연결하여 기계와 사람 모두 이해 가능한 시맨틱 웹을 구축했습니다. 의도 기반 자동완성/추천 검색 UX와 핵심 메타데이터 요약 카드로 탐색 경험을 강화했습니다.

Step 02

지식의 시각화 (Knowledge Graph)

복잡한 서지 정보를 직관적인 그래프 형태로 시각화하여, 정보의 관계를 한눈에 파악하고 탐색할 수 있습니다. 연관 데이터 간 이동을 고려한 Discovery Flow 설계로 노드(Node)와 링크(Link)가 연결된 지식 탐색 경험을 제공합니다.

Step 03

사용자 중심 검색 경험 (Optimized UX)

전문가만 쓰던 LOD 데이터를 일반인도 쉽게 검색하고 활용할 수 있도록, 직관적인 UI와 패싯 검색을 도입했습니다. 주제·형태·기관 등 다차원 필터 + 결과 수 카운트로 탐색 피드백을 강화하고, NL Blue 톤의 공공 신뢰감을 유지했습니다.

The Impact — 주요 성과

Accessibility Up

데이터 접근성 대폭 향상. 오프라인 서지자료 수백만 건을 온라인 데이터 1천만+ 건으로 확장, 누구나 검색·활용 가능

Standard Model

국가 서지 표준 모델 확립. 이후 공공 LOD 구축의 UX 레퍼런스로 활용

Visual Discovery

지식 탐색의 새로운 경험 제공. 사용자 피드백: "찾기 쉽다", "구조가 직관적이다" 등 긍정 반응 다수
