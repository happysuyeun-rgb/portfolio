공공기관 Project

국가서지 Linked Open Data (국립중앙도서관)

방대한 서지 데이터를 검색 · 활용하기 위한 플랫폼을 리뉴얼했습니다.
공공기관의 접근성/보안 기준을 충족시키면서도 사용자가 데이터를 ‘탐색’하는 경험을 중심으로 UI패턴을 설계했습니다.

#브랜딩 #브랜딩 #UI/UX #Design System

Project Overview

  • 기간: 2023년
  • 역할: UI/UX 총괄 · 컴포넌트 가이드 100% 전달
  • Site Concept: 데이터 탐색을 쉽게, 정보 구조를 명확하게 — 민간 서비스 수준의 사용성을 갖춘 공공 데이터 포털
  • Tools: Figma (디자인 시스템, 컴포넌트 문서화)

Problem

  • 방대한 텍스트 중심 화면 → 사용자 탐색 피로도 증가, 탐색 경로가 단절
  • 전문 용어와 구조가 개발자 중심 → 일반 사용자/연구자에게 난해
  • 공공기관 접근성 가이드로 현대적이고 직관적인 UI를 구현하기 어려움

Outcome

1. IA/검색 UX 개선
  • 의도 기반 자동완성/추천 검색 UX와 핵심 메타데이터 요약 카드 도입
  • 주제·형태·기관 등 다차원 필터 + 결과 수 카운트로 탐색 피드백 강화
  • 연관 데이터 간 이동을 고려한 탐색 흐름(Discovery Flow) 설계
2. 시각 체계/확장성 향상
  • NL Blue 톤의 공공 신뢰감 유지 + 현대적 정보 위계 정립
  • Figma 컴포넌트/레이아웃/모션 가이드 제공으로 운영·확장 효율 확보
  • 개발자용 OpenAPI/용어사전/개발 가이드 섹션 신설

Reflection

  • 방대한 DB를 ‘해석 가능한 정보’로 바꾸는 데 집중 — 메타데이터의 시각적 요약/우선순위화
  • 공공 제약 내에서도 패턴 라이브러리를 구축해 재사용 가능한 UX를 마련
  • 디자인 산출물을 Figma → 개발 가이드로 체계화해 협업 효율 향상

Impact & Value

  • 공공 포털임에도 깔끔하고 현대적인 브랜드 감각 확보 → 사용자 신뢰 상승
  • 사용자 피드백: “찾기 쉽다”, “구조가 직관적이다” 등 긍정 반응 다수
  • 오프라인 서지자료 수백만 건을 온라인 데이터 1천만+ 건으로 확장, 누구나 검색·활용 가능
  • 이후 공공 LOD 구축의 UX 레퍼런스로 활용
